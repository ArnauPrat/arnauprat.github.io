About Me and this Page

I computer scientist passionate about game engines and computer science in general. In this page you will find information the indie game I am currently working on, as well as other content related to game development and programming.

Indie Game

I am developing a rogue-like indie game in C99 and Vulkan almost from scratch. I use the following libraries though:
  • GLFW for window and input
  • CGLM for math
  • CIMGUI for the the development editor
  • CIMGUI for the the development editor
  • CGLTF for importing GLTF assets
  • STB_image and STB_truetype for image and font loading respectively
