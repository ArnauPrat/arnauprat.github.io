About Me and this Page

I am a computer scientist passionate about game engines and computer science in general. In this page you will find information about the indie game I am currently working on, as well as other content related to game development and programming.

Indie Game


I am developing an indie game in C99 and Vulkan almost from scratch. I use the following libraries though to assist in the development:
  • GLFW for window and input
  • CGLM for math
  • CIMGUI for the the development editor
  • CGLTF for importing GLTF assets
  • STB_image and STB_truetype for image and font loading respectively

The indie game is still at the early stages of development, but the goal is to create a replayable rogue-like game with a strategic yet real-time combat system.
